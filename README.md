This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Proyek Pembelajaran Next.js

Ini adalah repositori untuk mempelajari Next.js yang dibuat menggunakan perintah [`npm create-next-app@latest`](https://nextjs.org/docs/app/api-reference/cli/create-next-app). Proyek ini dikembangkan dengan menggunakan TypeScript dan Tailwind CSS untuk efisiensi pengembangan frontend.

## Fitur Utama

- **Next.js**: Framework React untuk pengembangan aplikasi web yang cepat dan efisien
- **TypeScript**: Bahasa pemrograman yang menambahkan tipe statis ke JavaScript
- **Tailwind CSS**: Framework CSS utility-first untuk styling yang cepat dan responsif

## Cara Menjalankan Proyek

1. Clone repositori ini ke lokal Anda:

```bash
git clone <repository-url>
```

2. Install dependencies:

```bash
npm install
```

3. Jalankan proyek:

```bash
npm run dev
```

4. Buka browser dan akses `http://localhost:3000` untuk melihat aplikasi berjalan.

## Struktur Proyek

- `src/app`: Berisi file-file halaman aplikasi
- `src/components`: Berisi komponen-komponen yang digunakan dalam aplikasi (masih kosong)
- `src/lib`: Berisi file-file library atau utilitas yang digunakan dalam aplikasi (masih kosong)
